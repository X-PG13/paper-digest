# Code of Conduct

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful in technical discussions.
- Accepting constructive feedback in good faith.
- Focusing on the problem, not the person.
- Sharing context clearly so others can reproduce issues.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Publishing private information without permission.
- Bad-faith disruption of technical discussions.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues,
and other contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull
requests, discussions, and any public communication on behalf of the project.
